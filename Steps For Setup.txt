Installing basics For Termux:
pkg update && pkg upgrade -y
pkg install proot-distro wget curl nano -y

Install Ubuntu:
proot-distro install ubuntu
proot-distro login ubuntu

Setup Ubuntu:
apt update && apt upgrade -y
apt install openjdk-21-jdk wget screen nano -y

Checking Java Version (It Must 21)
java -version

Installing Paper.MC
wget https://api.papermc.io/v2/projects/paper/versions/1.21.1/builds/133/downloads/paper-1.21.1-133.jar -O server.jar

Text File:
nano eula.txt(eula=true)

For Set up Play It.gg
cd ~

wget https://github.com/playit-cloud/playit-agent/releases/latest/download/playit-linux-aarch64

chmod +x playit-linux-aarch64

./playit-linux-aarch64

Starting Server:
Low Spec Phone:
java -Xms64M -Xmx192M -XX:+UseG1GC -jar server.jar nogui
Medium Spec Phone:
java -Xms256M -Xmx768M -XX:+UseG1GC -jar server.jar nogui
High Spec Phone:
java -Xms512M -Xmx1536M -XX:+UseG1GC -jar server.jar nogui

Download This For Starting And Stoping File To Store In Local
wget https://raw.githubusercontent.com/sriramanujamns-tech/MC-Server-On-Android/refs/heads/main/start.sh
wget https://raw.githubusercontent.com/sriramanujamns-tech/MC-Server-On-Android/refs/heads/main/stop.sh

chmod +x start.sh stop.sh

For Start And Stop Server
./start.sh
./stop.sh
