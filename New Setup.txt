Installing basics For Termux:
pkg update && pkg upgrade -y
pkg install proot-distro wget curl nano -y

Install Ubuntu:
proot-distro install ubuntu
proot-distro login ubuntu

Setup Ubuntu:
apt update && apt upgrade -y
apt install openjdk-21-jdk wget screen nano -y

Checking Java Version (It Must 21)
java -version

Installing Paper.MC
wget https://api.papermc.io/v2/projects/paper/versions/1.20.4/builds/386/downloads/paper-1.20.4-386.jar -O server.jar

Text File:
nano eula.txt(eula=true)

For Set up Play It.gg
cd ~

wget https://github.com/playit-cloud/playit-agent/releases/latest/download/playit-linux-aarch64

chmod +x playit-linux-aarch64

./playit-linux-aarch64

For Starting Server:
java -Xmx512M -Xms512M -jar server.jar nogui
